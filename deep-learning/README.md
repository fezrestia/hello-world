# Deep Learning from ZERO

## Env Setup

    $ apt install python3-venv
    $ python3 -m venv venv

    # activate venv
    $ source venv/bin/activate

    # deactivate venv
    $ deactivate

## Export PKG Dependency

    $ pip3 freeze > dependencies.txt

## Import PKG Dependency

    # after env setup
    $ pip3 install -r dependencies.txt
